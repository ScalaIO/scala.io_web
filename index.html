The Scala and FPL friendly event in Paris ! Tweet

October 24th and 25th, 2013. Paris, France

For Scala and functional programming

Scala.io is the event of the year in Paris about the Scala Programming language and Functional programming as a whole.

It will tackle not only Scala/JVM growing ecosystem but also functional programming languages like Haskell, Clojure, Erlang...

The venue is going to be awesome and Top Speakers and influencers of the Scala-tech world are already on their way.
